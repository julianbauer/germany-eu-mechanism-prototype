Wann eine Faszientherapie sinnvoll sein kann:

  • Sie spüren ein Ziehen im Rücken
  • Sie erleben erste Anzeichen von Fußbeschwerden
  • Sie können sich nicht mehr schnell bewegen
  • Sie bemerken eine Verschlechterung Ihrer allgemeinen gesundheitlichen Situation
